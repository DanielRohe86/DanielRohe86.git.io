Lista de Recursos CSS

Referente aos operadores e o que eles estilizam

  • font-size: relativa ao tamanho da fonte; pode ser proporcional ao tamanho da página;
  • font-weight: define a espessura da fonte;
  • text-decoration: define efeitos como sublinhado, ou com a linha cortando o texto;/li>
  • text-align: define o alinhamento do texto (centralizado, à esq. ou à dir);
  • line-height: define o espaçamento entre linhas;
  • letter-separation: define a separação entre letras, numa mesma linha;
  • color: define a cor da fonte;
  • background-color: define a cor de fundo
